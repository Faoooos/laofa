# TSgraph V1.1

We have provided more detailed code for TSgraph (Version 1.1). This is a transfer learning model used for time series classification and prediction.

The versions of the dependencies are as follows:
python=3.12
scikit-learn=1.7.1
scipy=1.16.0
scs=3.2.9
sentence-transformers=5.1.2
torch-geometric=2.7.0
torchaudio=2.5.1
torchvision=0.20.1
tornado=6.5.1
tqdm=4.67.1
transformers=4.49.0
tslearn=0.7.0

The Demo_data folder contains example data.

We recommend that users configure their own domain-specific large language models to run TSgraph.  Furthermore, to ensure optimal performance, we also suggest adjusting the model hyperparameters according to the specific domain.
